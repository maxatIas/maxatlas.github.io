# 404 Page — Design Spec

**Date:** 2026-04-14
**Status:** Approved

## Overview

Custom 404 page for the maxatlas Hugo site. Replaces the default Hugo 404 with a branded page that matches the cyberpunk/neon aesthetic and uses a retro arcade tone.

## Layout

**Split hero** — two-column layout:
- Left column (~42% width): cover image with scanline overlay (same treatment as article covers)
- Right column (flex: 1): error code + message + CTA button

**Mobile (≤ 600px):** stacks vertically — cover image on top (full width, fixed height), message block below.

## Copy

```
404
WRONG WARP PIPE

This level does not exist.
The plumber is somewhere, but not here.
No checkpoints. No respawn.

[← Find a valid warp]  →  links to /
```

- `404`: monospace, cyan neon glow (same as `.article__title`)
- `WRONG WARP PIPE`: monospace uppercase, magenta neon glow, smaller
- Body text: `#cbd5e1`, standard size
- CTA: monospace uppercase button, cyan border, links to `/`

## Cover Image

User-supplied. Place as `content/404/cover.*` (any extension — AVIF recommended).

**Image generation prompt:**
> Dark cyberpunk atmosphere, deep black and navy background. A glowing neon warp pipe — cyan and magenta neon light — spiraling into an endless void. The pipe is slightly bent, cracked, leading nowhere, surrounded by floating question marks and scattered pixel dust. Faint scanline overlay effect. Retro arcade meets cyberpunk aesthetic. Moody, slightly humorous. Portrait or square orientation, high contrast, no text.

## Template Structure (`404.html`)

Extends `baseof.html` via `{{ define "main" }}`. Reads `cover.*` from the `content/404/` page bundle using `.Resources.GetMatch "cover.*"`. If no cover is present, the split collapses: only the body column renders, full-width and centered.

```
baseof.html
  └── 404.html (defines "main")
        ├── .page-404 wrapper (max-width 680px, centered)
        │     ├── .page-404__hero (flex row)
        │     │     ├── .page-404__cover (42%, cover img + scanlines)
        │     │     └── .page-404__body (flex: 1)
        │     │           ├── .page-404__code  "404"
        │     │           ├── .page-404__sub   "WRONG WARP PIPE"
        │     │           ├── .page-404__msg   body text
        │     │           └── .page-404__cta   "← Find a valid warp"
        └── (mobile: cover stacks above body via media query)
```

## CSS Notes

New block `.page-404` added at the end of `main.css`. Reuses existing design tokens (`--accent-cyan`, `--accent-magenta`, `--glow-cyan`, etc.) and the `.article__cover-scanlines` pattern for the scanline overlay. No new tokens needed.

## Out of Scope

- Animated glitch effects
- Search box or suggested posts
- Translated copy (English only)
